# Email Digest

AI email filter that sends a daily digest to Telegram. Only the emails that matter.

Gmail IMAP + Gemini + Telegram bot. Runs at 10AM Nepal time. Cuts inbox noise to zero.

## Stack
- Python
- Gmail IMAP
- Google Gemini
- Telegram Bot API
